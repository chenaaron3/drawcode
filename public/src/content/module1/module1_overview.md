Welcome to programming! In this module, you'll learn the essential building blocks of Python programming.

## What You'll Learn

**Variables:** Store and work with different types of data

```python
name = "Alice"
age = 25
height = 5.6
```

**Operations:** Perform calculations and comparisons

```python
total = price * quantity
is_adult = age >= 18
```

**Input/Output:** Make programs interactive

```python
name = input("What's your name? ")
print(f"Hello, {name}!")
```

**Debugging:** Find and fix problems in your code

- Use the visual debugger to step through programs
- Watch variables change in real-time
- Understand error messages

## How to Use This Module

1. **Read the lesson content** on the left
