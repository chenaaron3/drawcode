# Operations: Working with Data

Operations let you manipulate and work with your variables.

## Arithmetic Operations

```python
a = 10
b = 3

sum_result = a + b       # Addition: 13
difference = a - b       # Subtraction: 7
product = a * b          # Multiplication: 30
quotient = a / b         # Division: 3.333...
remainder = a % b        # Modulo (remainder): 1
power = a ** b           # Exponentiation: 1000
```

## Comparison Operations

```python
x = 5
y = 8

print(x == y)    # Equal to: False
print(x != y)    # Not equal: True
print(x < y)     # Less than: True
print(x > y)     # Greater than: False
print(x <= 5)    # Less than or equal: True
print(y >= 8)    # Greater than or equal: True
```

## Logical Operations

```python
is_sunny = True
is_warm = False

print(is_sunny and is_warm)    # Both true: False
print(is_sunny or is_warm)     # At least one true: True
print(not is_sunny)            # Opposite: False
```

## Combining Operations

```python
age = 20
has_license = True

can_drive = age >= 18 and has_license  # True
price = 100
discount = 0.15
final_price = price * (1 - discount)   # 85.0
```
