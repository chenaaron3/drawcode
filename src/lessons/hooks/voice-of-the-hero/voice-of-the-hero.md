# Voice of the Hero

Now we're going to teach our program to communicate. The ability to speak is essential: a program can tell us many things about "how" or "why" or "what" it is doing. In Python, the `print()` function is used to make a program talk. The message to be printed should be surrounded by quotes:

```python
# A hero's battle cry echoes through the dungeon
print("Victory or death!")
```

In the above example, we direct our program to `print()` a heroic declaration. The printed words that appear as a result of the `print()` function are referred to as _output_. The output of this example program would be:

```
Victory or death!
```
