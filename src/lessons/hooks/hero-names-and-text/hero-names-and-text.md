# Hero Names and Text

In programming, we refer to blocks of text as **strings**. When you created your hero and displayed "Hello, Xaden!" in the previous lesson, you were working with strings. In Python, a string is text surrounded by either double quotes (`"Xaden"`) or single quotes (`'Xaden'`). The choice between single and double quotes doesn't affect functionality, but consistency in your code is important.

Whether you're storing a hero's name, a weapon description, or dialogue text, strings are fundamental to creating any RPG system in PythonQuest.

## Instructions

_Hint: Remember to put your name in quotes, like `"Xaden"` or `'Xaden'`_

### Checkpoint 2

**2.** If your print statement uses double quotes `"`, change them to single quotes `'`. If it uses single quotes `'`, change them to double quotes `"`.
