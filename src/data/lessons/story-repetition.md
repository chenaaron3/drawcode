# Story Repetition

Use loops to create dramatic tension in your storytelling! Repetition is a powerful literary device that builds suspense and atmosphere.

## Building Dramatic Tension

Create an ominous, repeating warning:

```python
warning_message = 'The ancient curse grows stronger'

print('=== THE APPROACHING DARKNESS ===')
print('')

for warning_count in range(3):
    print(warning_message + '...')
    
print('')
print('The final confrontation is near!')
```

## The Power of Repetition

Repeating the warning creates:
- **Building tension**: Each repetition increases the drama
- **Atmosphere**: The mood becomes more ominous
- **Anticipation**: Something big is about to happen
- **Emphasis**: The message becomes more memorable

## Predict Before You Run

Before executing, predict:
- How many times will the warning appear?
- What do the `...` add to the message?
- How does repetition change the feeling?

## Understanding `range(3)`

When you use `range(3)` without a starting number:
- It creates: 0, 1, 2
- The loop runs 3 times
- You don't always need to use the loop variable

## Literary Techniques in Code

Repetition in storytelling:
- **Rule of three**: Three repetitions feel complete
- **Escalation**: Each repetition can be slightly different
- **Rhythm**: Creates a memorable pattern
- **Foreshadowing**: Hints at important events

## Variations to Try

Enhance the drama by:
- Changing the message slightly each time
- Adding different punctuation (!, ?, ...)
- Using different numbers of repetitions
- Combining with other story elements

## Advanced Storytelling

In complex adventures, repetition might:
- Mark the passage of time
- Show a character's growing desperation
- Build to a climactic moment
- Create memorable catchphrases or prophecies

## Key Concept

Loops aren't just for functional tasks - they're powerful storytelling tools. Strategic repetition creates atmosphere, builds tension, and makes your adventure narratives more engaging.
