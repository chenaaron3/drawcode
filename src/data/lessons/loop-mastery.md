## Multiple Loop Types

Use different loop patterns in one program:

```python
print('=== THE GRAND ADVENTURE ===')
print('')

# Generate party members using list iteration
print('Your adventuring party:')
party_roles = ['warrior', 'mage', 'rogue']
for member in party_roles:
    print('- A skilled', member)

print('')

# Generate quest stages using range iteration
print('Your quest unfolds:')
for stage in range(1, 4):
    print('Stage', stage, ': The adventure deepens')
```

## Two Loop Styles

This program demonstrates both main loop types:

**List iteration**: `for member in party_roles`
- Loops through actual list items
- Perfect for predefined content

**Range iteration**: `for stage in range(1, 4)`  
- Loops through numbers
- Perfect for counting and numbering
