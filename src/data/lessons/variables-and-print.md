The `print()` function becomes much more powerful when you use it with variables. You can mix text and variable values to create meaningful output.

## Printing Variables with Labels

Instead of just printing a number:

```python
print(95)  # Just shows: 95
```

Add descriptive text:

```python
score = 95
print('Score:', score)  # Shows: Score: 95
```

## Multiple Items in print()

You can print several things at once by separating them with commas:

```python
print('Subject:', subject)
print('Score:', score)
```

## Important Points

- Separate items in `print()` with commas
- Python automatically adds spaces between items
- This makes your output much easier to understand
- Always label your values so people know what they mean

## Key Concept

Good programs explain what their output means. Using descriptive labels with your variables makes your program communicate clearly.
