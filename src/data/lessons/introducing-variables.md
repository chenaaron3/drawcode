Variables are containers that store information in your program. Think of them as labeled boxes that hold different types of data.

## Types of Data

Variables can store different types of information:

- **Numbers**: `age = 25`
- **Decimal numbers**: `height = 5.8`
- **Text**: `name = 'Bob'`
- **True/False**: `is_student = True`

## How Variables Work

```python
age = 25        # Store a number
name = 'Bob'    # Store text
```

The `=` sign means "store this value in this variable."

## Why Use Variables?

Variables let you:

- Remember information for later
- Give meaningful names to your data
- Reuse values throughout your program

## Key Concept

Variables are the foundation of programming. They let your program remember and work with information.
