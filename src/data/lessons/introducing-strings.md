Strings are how we work with text in Python. A **string** is simply text surrounded by quotes.

## Creating Strings

You can create strings using single quotes:

```python
name = 'Alice'
message = 'Hello, world!'
```

## Important Points

- Strings must be surrounded by quotes
- You can store strings in variables
- Even numbers can be strings if they're in quotes: `'123'`
- Strings are perfect for storing names, messages, and any text

## Key Concept

A string is text data that your program can work with. Think of it as storing words or sentences that your program can remember and use later.
