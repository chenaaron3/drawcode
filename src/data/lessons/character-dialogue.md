# Character Dialogue

Bring your characters to life with realistic dialogue! NPCs (Non-Player Characters) make adventures memorable through their conversations with the hero.

## Creating Character Speech

Store different types of dialogue:

```python
character_name = 'Old Sage'
greeting = 'Greetings, young adventurer!'
quest_hint = 'The treasure lies beyond the whispering caves.'
farewell = 'May fortune favor your journey.'
```

## Formatting Dialogue

Make dialogue look realistic with proper formatting:

```python
print(character_name + ': "' + greeting + '"')
print(character_name + ': "' + quest_hint + '"')
print(character_name + ': "' + farewell + '"')
```

## Predict Before You Run

Before executing, predict:
- How will the character name and dialogue combine?
- What do the quote marks (`"`) add to the dialogue?
- Which variable contains the quest clue?

## Dialogue Patterns

Notice the pattern: `character + ': "' + dialogue + '"'`
- Character name
- Colon and space
- Opening quote
- The actual words
- Closing quote
