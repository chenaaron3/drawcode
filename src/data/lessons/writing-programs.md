A program is a series of instructions that work together to accomplish a goal. Let's write a simple program that introduces a pet.

## Program Structure

Good programs follow a logical flow:

1. Set up your data (create variables)
2. Do something with the data
3. Show the results

## Example Program Flow

```python
# 1. Set up data
pet_name = 'Fluffy'
pet_type = 'cat'
pet_age = 3

# 2. Create output
print('Meet my pet!')
print('Name:', pet_name)
```

## Important Points

- Use meaningful variable names (`pet_name` not just `name`)
- Group related information together
- Add comments to explain what you're doing
- Think about what story your program tells

## Key Concept

Programming is about organizing information and instructions to solve problems or tell stories. Even simple programs should have a clear purpose and flow.

Watch how this program introduces a pet step by step, creating a complete story with just a few variables and print statements.
