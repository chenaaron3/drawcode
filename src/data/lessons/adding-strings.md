You can combine strings together using the `+` operator. This is called **string concatenation**.

## How String Addition Works

```python
first_name = 'Alice'
last_name = 'Smith'
full_name = first_name + ' ' + last_name
```

## Building Messages

String addition is perfect for creating personalized messages:

```python
greeting = 'Hello, ' + first_name + '!'
```

## Important Points

- Use `+` to join strings together
- Don't forget spaces: `' '` creates a space between words
- You can add as many strings as you want
- The result is a new, longer string

## Key Concept

String addition lets you build dynamic text by combining smaller pieces. This is how programs create personalized messages and formatted output.
