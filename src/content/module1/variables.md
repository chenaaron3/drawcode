# Variables: Storing Data

Variables are like labeled containers that store information in your program.

## Creating Variables

```python
name = "Alice"        # String (text)
age = 25             # Integer (whole number)
height = 5.6         # Float (decimal)
is_student = True    # Boolean (True/False)
```

## Using Variables

```python
# Use variables in calculations
birth_year = 2024 - age
greeting = "Hello, " + name

# Variables can change
age = age + 1        # Now 26
```

## Naming Rules

- Start with letter or underscore: `name`, `_count`
- Use letters, numbers, underscores: `user_age`, `score2`
- Case-sensitive: `Name` ≠ `name`
- No spaces: use `first_name` not `first name`
